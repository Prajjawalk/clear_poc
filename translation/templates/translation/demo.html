Translation Package Features:
  • Store static strings in database
  • Multi-language support via django-modeltranslation
  • Template tags for easy integration
  • Robust fallback behavior
  • Caching for performance
  • Parameter substitution

Template Tag Examples

Basic Usage

Template: {% verbatim %}{% translate "welcome_message" %}{% endverbatim %}

Result: {% translate "welcome_message" %}

With Parameters

Template: {% verbatim %}{% translate "hello_user" name="Alice" %}{% endverbatim %}

Result: {% translate "hello_user" name="Alice" %}

Filter Usage

Template: {% verbatim %}{{ "button_save"|t }}{% endverbatim %}

Result: {{ "button_save"|t }}

Fallback Behavior

Template: {% verbatim %}{% translate "nonexistent_label" %}{% endverbatim %}

Result: {% translate "nonexistent_label" %}

When a translation doesn't exist, the label itself is returned.
