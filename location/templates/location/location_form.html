Guidelines
Administrative Hierarchy
  • Country (Level 0): Root level, no parent
  • State (Level 1): Parent must be Country
  • District (Level 2): Parent must be State
  • County (Level 3): Parent must be District
Geographic ID Format

Use hierarchical format like:

  • SD (for Sudan - Country)
  • SD_001 (for state under Sudan)
  • SD_001_002 (for district under state)
Geographic Data

Boundary and point data are optional but useful for:

  • Mapping and visualization
  • Spatial analysis
  • Geographic searches
