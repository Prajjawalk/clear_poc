What is a Gazetteer?

A gazetteer is a directory of alternative names and codes used by different data sources to refer to the same location. This helps with data integration and location matching.

Examples

Location: Khartoum State

  • ACLED: "Khartoum"
  • UNHCR: "Khartoum State"
  • Local: "ولاية الخرطوم"
Common Sources
  • ACLED: Armed Conflict Location & Event Data
  • UNHCR: UN High Commissioner for Refugees
  • OCHA: UN Office for Coordination of Humanitarian Affairs
  • local: Local administrative names
  • ISO: International Organization for Standardization codes
Tips
  • Use consistent source identifiers
  • Include both codes and names when available
  • Add entries for all variations used by data sources
