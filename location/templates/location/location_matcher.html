How Location Matching Works

The location matcher uses exact matching to find locations in the database. It searches through:

  1. Gazetteer entries - Alternative names from various sources
  2. Location names - Official location names
Matching Strategy
  • Exact name matching only (case-insensitive)
  • No fuzzy matching to avoid false positives
  • Source-specific matching when source is provided
  • Administrative level filtering
  • Parent context for disambiguation
Tips for Better Matching
  • Specify the source if you know it
  • Use parent context to disambiguate common names
  • Check different spelling variations
  • Consider administrative level constraints
